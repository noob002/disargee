v1.0 - 초기 컨셉 (Prototype)
- 핵심 로직 구현: "다수결의 반대"인 소수결(Minority) 게임 룰 정립.

- 승리 조건: 전체 선택 비율 중 50% 미만인 쪽을 선택해야 생존.

- 기본 UI: HTML/CSS를 활용한 심플한 버튼형 웹게임 구조.

v2.0 - 레트로 비주얼 업데이트 (Visual Overhaul)
- 테마 변경: 90년대 DOS/CRT 모니터 감성 적용 (초록색 터미널 폰트 VT323).

- 시각 효과: 화면 스캔라인(Scanline), 텍스트 타이핑 효과, 게이지 바 애니메이션 추가.

- 분위기: 어둡고 불길한 SF/디스토피아 분위기 조성.

v3.0 - 다단계 스테이지 확장 (Multi-Phase)
- Phase 1 (심리): 기존 소수결 게임 유지.

- Phase 2 (해킹): 제한 시간 내 코드를 입력하는 타이핑 디펜스 추가. (긴장감 조성)

- Phase 3 (생존): 마우스로 떨어지는 물체를 피하는 탄막 회피(Dodge) 게임 추가.

v3.1 - Phase 3 전투 시스템 개편 (Combat Update)
- 경고 시스템(Telegraph) 추가: 공격 전 **'노란색 점선'**이 깜빡이며 공격 위치를 미리 알려줌.

- 패턴 이원화:

- 예고 공격: 점선 경고 후 빠르게 낙하.

- 기습 공격: 경고 없이 느리게 낙하.

- 단계별 난이도: 초록(안전) → 노랑(경고) → 빨강(위험)으로 배경색과 난이도가 시간에 따라 변화.

- 휴식 구간(Break Time): 난이도 변경 전 5초간 휴식 및 스토리 텍스트("당신은 과연...") 출력.

v3.2 - 편의성 및 엔딩 강화 (QoL & Polish)
- 엔딩 연출: 클리어 시 화려한 픽셀 폭죽 애니메이션과 네온 사인 텍스트 추가.

- 난이도 조절(Nerf): 마우스 피격 판정(Hitbox) 축소 및 적 낙하 속도 대폭 하향.

- 스킵(Skip) 기능: 5회 이상 게임 오버 시, [엔딩으로 건너뛰기] 버튼 활성화 (로컬 저장소 기능 활용).

- 멀티 엔딩: 직접 깬 'LEGEND' 엔딩과 스킵한 'SKIPPED' 엔딩 구분.

v4.0 - 악몽 모드 (Nightmare Update)
- 히든 콘텐츠: 노말 모드를 1회 클리어한 유저에게만 [☠ NIGHTMARE ☠] 버튼 해금.

- 카오스(Chaos) 패턴: 악몽 모드 30초 이하 진입 시, 게임 화면 자체가 윈도우 화면보호기처럼 벽에 튕기며 이동.

- 전용 테마: 보라색(Purple) & 청록색(Cyan) 글리치 효과 적용.

- 칭호: 악몽 모드 클리어 시 "GOD MODE" 칭호 획득.

v4.1 - 밸런스 최종 패치 (Final Balance)
- 악몽 모드 너프:

- 화면 이동 속도 50% 감소 (따라가기 쉬워짐).

- 탄막 속도 하향 (반응 가능한 수준으로 조정).

- 경고 시간 증가 (피할 시간 확보).

- 최적화: 카오스 모드 진입 시 프레임 드랍 방지 및 UI 위치 재정렬.
